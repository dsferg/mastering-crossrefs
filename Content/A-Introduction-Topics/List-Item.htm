Referencing a list item

In order to reference a specific item in a list, the list items must be numbered using Flare's autonumbering feature. You can either create autonumbered paragraphs to accomplish this, or you can add autonumbering to your list items in your ordered lists.

Ordered lists

In order to use ordered lists and still be able to reference step numbers, you'll need to set your list items to be numbered using Flare's autonunmbering feature. In this sample project, here is the CSS for the list item autonumbering:

/*select the first list item in a, ordered list and number it step 1*/
ol li:nth-child(1){
	list-style-type: none;
	mc-auto-number-format: 'L:{n=1}. ';
}

/*select the subsequent list items in an ordered list and increment the number*/
ol li{
	list-style-type: none;
	mc-auto-number-format: 'L:{n+}. ';
}

Next, create an xref class for list item references:

MadCap|xref.StepNumber{
	mc-format: '{paranumonly}';
}
			
@media print{
	MadCap|xref.StepNumber{
		mc-format: '{paranumonly} ({pageref})';
	}
}
